So — what did I build with it?

A real-time translation overlay for Windows. Written in Go. ~95% of the first draft came from a local model.

The overlay

  • Always-on-top transparent window, drag-to-move, remembers its position.
  • Streams partial translations as they arrive — no awkward waiting for a full sentence.
  • Bilingual mode: original + translation stacked, so you can sanity-check the model live.

Main overlay — compact, dark, unobtrusive.

Pick your translator

  • Ollama / LM Studio — 100% local, zero cost, zero leakage.
  • Google free — quick, no key.
  • DeepL — best quality for EU languages, key-based.
  • Optional glossary + streaming chat-completions for tone control.

Translation providers — swap in one click.

Captions tuned to the speaker

  • Source language hints, VAD sensitivity, silence thresholds.
  • Whisper tiers from tiny to large-v3 — trade latency for accuracy.
  • Works on loopback audio — captures whatever plays through your speakers (Teams, browser, media players).

Captions & ASR configuration.

Readable at 3 a.m.

  • Dark theme, DPI-aware (PerMonitorV2), tuned for 125–200% scaling.
  • Font size, opacity, and bilingual layout all adjustable.
  • Polish / English UI — i18n baked in, easy to extend.

Appearance — dark, readable, configurable.

The takeaway

  • Don't let AI own your critical path — payments, auth, infra-as-code, anything that can page you at 3 a.m.
  • Do let it build the 100 tiny tools around your critical path — overlays, converters, internal dashboards, one-shot scripts.
  • The leverage is enormous. This whole program is a weekend of glue code I would never have written by hand.

AI as a force multiplier for peripheral tooling — that's where the safe ROI lives today.
